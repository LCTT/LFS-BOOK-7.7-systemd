6.33. Bison-3.0.4

Bison 软件包包含一个语法生成器。

大概编译时间： 2.3 SBU
需要磁盘空间： 40 MB

6.33.1. 安装 Bison

准备编译 Bison： 

./configure --prefix=/usr --docdir=/usr/share/doc/bison-3.0.4

编译软件包： 

make

用以下命令测试结果(大约 0.5 SBU)： 

make check

安装软件包： 

make install

6.33.2. Bison 软件包内容

安装的程序： bison 和 yacc
安装的库： liby.a
安装目录： /usr/share/bison

简要介绍

bison

根据一系列规则生成用于分析文本结构的程序；Bison 是 Yacc(Yet Another Compiler Compiler) 的替代品。

yacc

bison 的封装，用于仍然调用 yacc 而不是 bison 的程序；它用 -y 选项调用 bison

liby

Yacc 库包含和 Yacc 兼容的 yyerror 和 main 程序的实现；这个库并不是很有用，但是 POSIX 要求要有它
