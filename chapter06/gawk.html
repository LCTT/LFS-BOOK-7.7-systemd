6.47. Gawk-4.1.1

The Gawk package contains programs for manipulating text files.

Approximate build time: 0.3 SBU
Required disk space: 39 MB

6.47.1. Installation of Gawk

Prepare Gawk for compilation: 

./configure --prefix=/usr

Compile the package: 

make

To test the results, issue: 

make check

Install the package: 

make install

If desired, install the documentation: 

mkdir -v /usr/share/doc/gawk-4.1.1
cp    -v doc/{awkforai.txt,*.{eps,pdf,jpg}} /usr/share/doc/gawk-4.1.1

6.47.2. Contents of Gawk

Installed programs: awk (link to gawk), gawk, gawk-4.1.1, and igawk
Installed libraries: filefuncs.so, fnmatch.so, fork.so, inplace.so, ordchr.so, readdir.so, readfile.so, revoutput.so, revtwoway.so, rwarray.so, testext.so, and time.so
Installed directories: /usr/lib/gawk, /usr/libexec/awk, /usr/share/awk, and /usr/share/doc/gawk-4.1.1

Short Descriptions

awk

A link to gawk

gawk

A program for manipulating text files; it is the GNU implementation of awk

gawk-4.1.1

A hard link to gawk

igawk

Gives gawk the ability to include files
