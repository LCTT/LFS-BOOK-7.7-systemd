v. 本书中的软件包逻辑

正如之前所述，LFS的目的是构建一个完整可用的基础级系统。这包括 基于用户的选择定制一个更加完整的系统提供相对最小基础而需要自我 复制的所有软件包。这并不意味着LFS是可能的最小系统。它也包括一些 并不严格要求的重要软件包。下面的列表介绍了本书中每个软件包需要 的理由。

  • Acl

    这个软件包包括管理访问控制列表的工具，用于定义文件和目录 更细粒度的自主访问权。

  • Attr

    这个软件包包括管理文件系统对象扩展属性的程序。

  • Autoconf

    这个软件包包括能生成根据开发者的模板自动配置源代码的shell脚本 的程序。更新了构建过程之后重新构建软件包通常需要这个。

  • Automake

    这个软件包包括从模板生成Make文件的程序。更新了构建过程之后重新 构建软件包通常需要这个。

  • Bash

    这个软件包能满足提供到系统的Bourne Shell接口的LSB核心需求。由于 通用性和能在基本shell功能上进行扩展，它能取代其它的shell软件包。

  • Bc

    这个软件包提供了能处理任意精度数值的语言。它能满足构建Linux内核 时的需求。

  • Binutils

    这个软件包包括一个链接器，汇编器和其它处理对象文件的工具。 编译LFS系统以及之上的大部分软件包需要这个软件包中的程序。

  • Bison

    这个软件包中有yacc(Yet Another Compiler Compiler)的GNU版本， 需要用来构建一些其它的LFS程序。

  • Bzip2

    这个软件包包括用来压缩和解压缩文件的程序。在解压缩很多LFS 软件包的时候需要它。

  • Check

    这个软件包包括一个用于其它程序从测试工具。它只安装在临时工具链中。

  • Coreutils

    这个软件包包含一些查看和管理文件和目录的重要程序。在命令行 文件管理和每个LFS软件包的安装过程中需要它。

  • D-Bus

    这个软件包包括一些实现一个消息总线的程序，消息总线是一种应用 程序和另一个应用程序通信的简单方式。systemd也需要它。

  • DejaGNU

    这个软件包包括一个测试其它程序的框架。只安装在临时工具链中。

  • Diffutils

    这个软件包包括一些显示文件和目录差别的程序。这些程序用来 创建补丁，以及很多软件包的构建过程。

  • E2fsprogs

    这个软件包包括一些处理ext2，ext3和ext4文件系统的工具。这些是 Linux支持的最常用而且完全通过测试的文件系统。

  • Expat

    这个软件包包括一个相对小的XML解析库。Perl模块XML::Parser需要用到这个。

  • Expect

    这个软件包包括一个执行和其它交互程序交互的脚本对话框。 通常用来测试其它软件包。只安装在临时工具链中。

  • File

    这个软件包包括一个能判断给定文件文件类型的工具。 一些软件包需要用它来构建。

  • Findutils

    这个软件包包括一些在文件系统中查找文件的程序。 在很多软件包构建脚本中会用到它。

  • Flex

    这个软件包包括一个能生成识别文本模式程序的工具。是lex(lexical analyzer) 程序的GNU版本。构建很多LFS软件包需要用到它。

  • Gawk

    这个软件包包括一些操作文本文件的程序。是awk(Aho-Weinberg-Kernighan) 的GNU版本。在很多软件包的构建脚本中会用到它。

  • Gcc

    这个软件包是GNU编译器工具集。它包括C和C++的编译器以及其它一些不是由LFS构建的。

  • GDBM

    这个软件包包含数据库管理库。LFS的另一个软件包Man-DB会用到它。

  • Gettext

    这个软件包包括实现很多软件包国际化和本地化需要用到的工具和库。

  • Glibc

    这个软件包包括主要的C语言库。缺少它Linux程序就运行不了。

  • GMP

    这个软件包包括能提供任意精度数值运算的数学库。编译Gcc会用到它。

  • Gperf

    这个软件包包括一个能从一个键集生成完美哈希函数的程序。sysetmd会用到它。

  • Grep

    这个软件包包括一些在文件中搜索的程序。大部分软件包的构建脚本会用到它。

  • Groff

    这个软件包包括处理和格式化文本的程序。其中一个重要的功能是格式化man页面。

  • GRUB

    这个包是Grand Unified Boot Loader。是可用的boot加载器之一，但最灵活。

  • Gzip

    这个包包括一些压缩和解压缩文件的程序。解压很多LFS以及之上的软件包时会需要它。

  • Iana-etc

    这个软件包为网络服务和协议提供数据。启用合适的网络功能会用到它。

  • Inetutils

    这个包包括基本网络管理的程序。

  • Intltool

    这个包包括能从源文件中抽取可翻译字符串的工具。

  • IProute2

    这个包包括一些基本和高级的IPv4和IPv6网络的程序。由于它的IPv6功能， 比起其它普通网络工具通常选择它。

  • Kbd

    这个包包括一些键表文件，用于非US键盘的键盘工具以及一些控制台字体。

  • Kmod

    这个软件包包括一些管理Linux内核模块的程序。

  • Less

    这个软件包包括一个很好的文本文件查看器，运行查看文件的时候向上或向下滚动。 也用于Man-DB查看man页面。

  • Libcap

    这个包实现了Linux内核中可用的从用户空间到POSIX 1003.1e的接口。

  • Libpipeline

    Libpipeline软件包包括一个以灵活和便捷的方式操作子进程流水线的库。 Man-DB软件包会用到它。

  • Libtool

    这个软件包包括一些GNU通用库支持脚本。它降低了在一致，可移植 的接口上使用共享库的复杂度。在其他LFS软件包的测试套件里需要它。

  • Linux Kernel

    这个包是操作系统。是GNU/Linux环境中的Linux。

  • M4

    这个包包括一个普通的文本宏处理器，作为其它程序的构建工具使用。

  • Make

    这个包包括一个指导软件包构建的程序。LFS中的几乎每个包都需要它。

  • Man-DB

    这个包包括一些查找和查看man页面的程序。由于更好的国际化功能， 用来代替man软件包。它支持man程序。

  • Man-pages

    这个包包括基本Linux man页面的真正内容。

  • MPC

    这个包包括复数运算的函数。Gcc需要它。

  • MPFR

    这个包包括多精度运算的函数。Gcc需要它。

  • Ncurses

    这个包包括一些处理字符界面的终端无关的库。通常用来为菜单系统 提供光标控制。一些LFS的软件包会用到它。

  • Patch

    这个包包括一个通过使用通常由diff 程序创建的补丁 文件来修改或新建文件的程序。一些LFS软件包的构建过程会需要它。

  • Perl

    这个包包括一个运行时语言PERL的解析器。一些LFS软件包的安装和测试套件会需要它。

  • Pkg-config

    这个包提供了一个返回已安装库或软件包元数据的程序。

  • Procps-NG

    这个包包括了一些监视进程的程序。这些程序对系统管理非常有用，也 用于LFS的boot脚本。

  • Psmisc

    这个包包括一些显示运行中进程信息的程序。这些程序对系统管理非常有用。

  • Readline

    这个包提供了一些命令行编辑和历史功能的库。Bash会使用它。

  • Sed

    这个包允许不在一个文本编辑器中打开而直接编辑文本。大部分LFS 软件包的配置脚本需要它。

  • Shadow

    这个包包括一些以安全方式处理密码的程序。

  • Systemd

    作为Sysvinit的替代品，这个包提供了一个init 程序以及一些其它的boot和系统控制功能。很多Linux发行版都用它。

  • Tar

    这个软件包提供的LFS使用的几乎所有的包归档和提取功能。

  • Tcl

    这个包包括了在很多LFS软件包测试套件中使用的工具命令语言。 只安装在临时工具链中。

  • Texinfo

    这个包包括一些读，写以及转换信息页面的程序。在很多LFS软件包的 安装过程中会使用它。

  • Util-linux

    该软件包包含许多工具。其中有处理文件系统，控制台，分区和消息的工具。

  • Vim

    这个包包括一个编辑器。由于vi编辑器的经典以及大量的强大功能而选择它。 对很多用户来说一个编辑器是一个非常个人的选择，如果需要的话其 它编辑器也能替代它。

  • XML::Parser

    这个包是和Expat交互的Perl模块。

  • XZ Utils

    这个包包括一些压缩和解压缩文件的程序。它通常提供最高的压缩率以及在解压XZ或者 LZMA格式的软件包时非常有用。

  • Zlib

    这个包包括一些程序使用的压缩和解压缩例程。
